# Respeitar é Proteger — Conscientização contra a Violência à Pessoa Idosa

Site informativo desenvolvido como trabalho acadêmico de Front-End, com o
objetivo de conscientizar sobre a violência contra a pessoa idosa: suas formas,
os sinais de alerta e os canais oficiais de denúncia e apoio.

🔗 **Acesse o site:** https://assuncaoguilherme23-ship-it.github.io/Conscientiza-o-contra-a-Viol-ncia-Pessoa-Idosa/

## Sobre o projeto

O portal é composto por três páginas que abordam o tema de forma clara e
acessível, reunindo informações para reconhecer, prevenir e combater a violência
contra idosos. A data de referência é o Dia Mundial de Conscientização contra a
Violência à Pessoa Idosa (15 de junho).

## Páginas

- **Início** (`index.html`) — apresentação do tema, missão do portal e
  estatísticas sobre a violência contra a pessoa idosa.
- **A Violência** (`informacoes.html`) — os tipos de violência, sinais de alerta
  e um quiz interativo para testar o conhecimento.
- **Como Ajudar** (`apoio.html`) — canais oficiais de denúncia e orientações
  sobre como agir com segurança.

## Tecnologias

- **HTML** — estrutura e conteúdo semântico das páginas
- **CSS** — estilização, layout responsivo e fontes locais (Inter e Newsreader)
- **JavaScript** — interatividade, como o quiz e a animação das estatísticas

## Estrutura de arquivos

```
.
├── index.html          # Página inicial
├── informacoes.html    # Tipos de violência e quiz
├── apoio.html          # Canais de denúncia e apoio
├── css/
│   └── estilo.css      # Estilos do site
├── js/
│   └── script.js       # Interatividade
└── fonts/              # Fontes locais (Inter, Newsreader)
```

## Como executar localmente

Por ser um site estático, não há etapa de build. Basta clonar ou baixar o
repositório e abrir o arquivo `index.html` no navegador.
